## 【特徴】  
- 鉄鋼業をイメージするグレーをベースにしています。  
- 6秒ごとに1.5秒かけて等速でスライドする画像を入れています。  
- hoverした際の変化を分かりやすくするために背景色を入れたり、画像が少し大きくなるを取り入れました。  
- レスポンシブ対応は、1000px以下でハンバーガーメニューに変更したり、CONTACTとQ&Aの2カラムを1カラムにして、スマホでも読みやすくしています。
